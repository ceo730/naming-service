# 작명 보고서 프리미엄 업그레이드 설계

## 목표
- 현재 "10~15만원짜리를 28만원에 파는 구조" → **49만원 가치의 보고서**로 개선
- 핵심: 개인화 충격, 선택 설득 구조, 프리미엄 의식(ritual) 연출

## 변경 범위

### 1. saju_engine.py — 십신/대운 계산 추가

#### 1-A. 십신(十神) 분석
일간(日干)을 기준으로 나머지 7자(+장간)와의 관계를 십신으로 분류:

| 관계 | 같은 음양 | 다른 음양 |
|------|-----------|-----------|
| 같은 오행 | 비견(比肩) | 겁재(劫財) |
| 내가 생하는 | 식신(食神) | 상관(傷官) |
| 내가 극하는 | 편재(偏財) | 정재(正財) |
| 나를 극하는 | 편관(偏官) | 정관(正官) |
| 나를 생하는 | 편인(偏印) | 정인(正印) |

**구현:**
```python
def calculate_sipsin(saju_result):
    """사주 8자 + 장간에 대해 십신 분류 및 카운트 반환"""
    # Returns: {
    #   'sipsin_count': {'비견': 1, '식신': 2, ...},
    #   'sipsin_detail': [{'position': '년간', 'sipsin': '편인', ...}, ...],
    #   'personality_traits': [...],  # 십신 조합으로 도출된 성격 키워드
    #   'career_aptitude': [...],
    #   'relationship_pattern': '...',
    # }
```

**성격 매핑 테이블:**
- 비견 강: 독립심, 자존심, 동료의식 → "내 길을 가는 사람"
- 식신 강: 표현력, 풍류, 먹복 → "창의적이고 감각적인 사람"
- 정재 강: 성실, 절약, 책임감 → "믿음직하고 안정적인 사람"
- 편관 강: 리더십, 결단력, 엄격 → "카리스마 있는 실행가"
- 정인 강: 학구적, 관대, 체면 → "지적이고 품격 있는 사람"
- 등등 10가지 각각에 대한 성격 3줄 + 대인관계 패턴 + 직업적성

**핵심 규칙:**
- 십신 카운트가 2개 이상인 것 = "두드러진 특성" → 이것을 충격 포인트로 활용
- 0개인 십신 = "부족한 영역" → 이름으로 보완해야 하는 포인트

#### 1-B. 대운(大運) 계산
월주(月柱)에서 순행/역행으로 10년 단위 운의 흐름 계산:

**알고리즘:**
1. 성별 + 년간 음양 → 순행/역행 결정
   - (남자+양간) 또는 (여자+음간) → 순행
   - (남자+음간) 또는 (여자+양간) → 역행
2. 대운 시작 나이 계산: 생일~다음(또는 이전) 절기까지 일수 ÷ 3 = 대운수
3. 월주 간지에서 순행/역행으로 천간+지지를 하나씩 이동

**구현:**
```python
def calculate_daeun(saju_result, gender, birth_year, birth_month, birth_day):
    """대운 계산 — 10년 단위 운의 흐름"""
    # Returns: {
    #   'direction': '순행' or '역행',
    #   'start_age': 3,  # 대운 시작 나이
    #   'periods': [
    #       {'age_start': 3, 'age_end': 12, 'stem': '庚', 'branch': '申',
    #        'ohaeng': '金', 'description': '...', 'key_events': '...'},
    #       ...
    #   ],
    #   'current_period': {...},  # 현재 대운 (나이 기준)
    #   'past_turning_points': [...],  # 과거 전환점
    #   'future_turning_points': [...],  # 미래 전환점
    # }
```

**전환점 판별:**
- 대운 천간/지지가 사주 원국과 충(沖)이면 → "큰 변화의 시기"
- 대운이 용신 오행이면 → "좋은 기회의 시기"
- 대운이 기신(忌神) 오행이면 → "주의가 필요한 시기"

#### 2-C. 섹션별 프롬프트 상세

**섹션 2: 성격 심층 진단**
- 십신 카운트 데이터를 프롬프트에 직접 주입
- "식신이 2개로 두드러진 사주입니다. 이것이 의미하는 구체적 성격 특성 3가지를 서술하되, 각 특성이 일상에서 어떻게 발현되는지 구체적 상황 예시를 포함하십시오"
- 대인관계: "이 십신 구조의 사람이 보이는 전형적 관계 패턴을 2가지 서술"
- 직업적성: "이 사주의 십신 구조가 특히 강점을 보이는 직업 영역 3가지와 그 이유"

**섹션 3: 인생 흐름 예측**
- 대운 데이터를 프롬프트에 직접 주입
- 과거 대운에서: "~세 전후로 중요한 변화나 전환이 있었을 가능성이 높습니다. 그 시기의 대운이 [오행]으로 사주 원국의 [오행]과 충을 이루어..."
- 미래 대운에서: "향후 ~세경에 [오행] 대운이 들어오면서 [용신/기신]과의 작용으로..."

**섹션 4-6: 이름별 시나리오 분석 (핵심 변경)**
기존 구조: 설명 → 장점 나열
새 구조:
1. 이 이름과 함께할 구체적 미래상 (사주 보완 효과가 인생에 미칠 영향)
2. 이 이름이 가져올 구체적 변화 시나리오 (대운과 연계)
3. 이 이름만의 고유한 강점 (다른 후보에는 없는 것)
4. 이 이름을 선택하지 않았을 때 놓치게 되는 것

**섹션 7: 최종 비교 (선택 압박 구조)**
- 각 이름을 선택했을 때의 미래 시나리오를 대비
- "A를 선택하면... B를 선택하면... C를 선택하면..." 구조
- 최종 추천에 감정적 임팩트 추가: "이 이름이 가져올 변화를 생각하면..."

**섹션 8: 이름 사용 가이드 (NEW)**
프롬프트 구조:
- 개명 최적 시기 (대운/세운 기반)
- 이름 첫 사용 의식: 공적 문서 변경 순서, 첫 서명 의미
- 주변 알림 전략: 가족 → 직장 → 공적 순서
- 금기사항: 이 사주에서 피해야 할 행동/방향
- 변화 흐름 예측: 30일 / 3개월 / 1년 / 3년 후 기대 변화
- 이름의 에너지를 극대화하는 생활 습관

## 리스크 & 완화

- **API 비용 증가**: 5회 → 8회 (60% 증가). 그러나 가격이 28→49만원이면 충분히 정당화
- **대운 계산 정확도**: 절기 기반 계산이 이미 있으므로 이를 활용. 대운 시작 나이는 생일~절기 일수/3으로 근사
- **프롬프트 길이**: 십신/대운 컨텍스트 추가로 프롬프트 길이 증가 → max_tokens 조정 필요
- **챕터 오프닝 이미지**: 현재 5개만 있음 → 순환 재활용으로 해결
